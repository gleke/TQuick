# TQuick

#### 介绍
一款基于Qt(Quick)跨平台技术采用自身模块规范编写的轻量级QML UI框架（基于Toou团队开发的Toou-2d框架开发），遵循Qt书写与组织形式门槛极低无需深入学习简单易用可拿来即用，丰富的控件模块适合界面的快速开发，让程序人员拥有更多的精力来实现业务逻辑与算法。
